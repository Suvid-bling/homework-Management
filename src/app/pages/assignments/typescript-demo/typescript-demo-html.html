使用提示

  • TypeScript 是 JavaScript 的超集，添加了类型系统和其他特性
  • 使用类型注解可以提高代码的可维护性和可读性
  • TypeScript 的类型系统在编译时进行检查，运行时会被擦除
  • 合理使用 any 类型，尽量使用更具体的类型
  • 善用联合类型和交叉类型来构建复杂的类型结构
